# Roadmap

- Local file capture.
- Explicit URL fetch capture.
- Markdown, JSON, and oss-ideas exports.
- Fixture-backed parser and CLI tests.

## Next

- Citation spans with line offsets.
- Optional PDF text adapter.
- Better HTML readability extraction without hidden services.
- Browser extension prototype that hands content to the local CLI.

## Later

- Desktop drop target.
- Pluggable transcript importers.
- Research packet diffing for updated sources.
